1) ul (Unordered list {data with bullet points (.)})

We can make changes in the style of bullet points(square,circle,disc,triangle) by making changes in css 

We can use "type" attribute to give square,circle,disc, triangle shape to the bullet points in ul.
        Note : triangle is not supported in all browsers
  • Apple.
    • color : Red.
    • season : In winter.
  • Mango.
    • color : Yellow.
    • season : In summer.

2) ol (ordered list {data will be in some order (1,2...) or (a,b,c...) or (roman numbers)})

We can make changes in the style of order(numerical,roman numbers,alphabets) by making changes in css 

We can use "type" attribute to give alphabets(cap, small) numeral, roman number in ol.
  1. Apple.
    1. color : Red.
    2. season : In winters.
  2. Mango
    1. color : Yellow.
    2. season : In summer.
